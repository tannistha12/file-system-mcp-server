# Product Requirements Document (PRD)

## Project
File System MCP Server

## Goal
Build an MCP server that exposes safe file-system tools to an AI client.

## Core Features
- List directories
- Read files
- Write files
- Search files
- Create folders
- Safety checks (restrict access to allowed workspace)

## Non-Goals
- GUI
- Database
- Authentication (v1)

## Success Criteria
The AI can discover and invoke every registered tool successfully.
